About This Project

This SPA demonstrates the core concepts of single-page applications:

  • Client-side routing using URL hash
  • Dynamic content switching without page reloads
  • Smooth transitions between views
  • Responsive design
  • Integrated API with Azure Functions

Technologies Used

This project uses vanilla web technologies - HTML5, CSS3, and JavaScript (ES6+). The backend uses Azure Functions for serverless API endpoints. No dependencies or build tools required!
